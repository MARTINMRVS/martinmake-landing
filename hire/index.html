How I work.

Structured AI-native workflow. Every build follows a documented process with specialized agents for each phase.

01 ARKI

Architecture first.

Before writing a single line, I plan impact: which files change, what could break, and the minimum viable approach. If it touches more than 2 files, architecture comes before execution.

02 DECK

AI-powered execution.

Claude Code as the primary executor. Skills and agents handle code generation, iteration, and deployment. I direct what to build, the system handles how — with full review at every step.

03 QATEST

Verify before ship.

Automated quality checks before every build. Clean code, no regressions, edge cases covered, UX validated. 12+ custom skills ensure consistent output across every project.
